# zhsite-aiyouxi-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可以用于展示、测试、存档或其他通用用途。仓库不针对任何特定域名或网站，仅作为静态 HTML 内容的集合与发布平台。

## 目录说明

- `docs/` —— 存放所有 HTML 页面文件及相关静态资源（如 CSS、JS、图片等）。该目录也是 GitHub Pages 的发布源。
- `README.md` —— 本文件，仓库说明。

## 页面归档说明

- 每个 HTML 页面均以独立文件形式存放，文件名清晰标识内容或用途。
- 页面之间互不依赖，可单独访问。
- 部分页面可能包含外部资源引用（如字体、库文件），请确保引用来源稳定或已本地化。
- 归档内容不涉及商业、营销或特定推广信息。

## 维护说明

- 新增页面请放入 `docs/` 目录，并保持命名规范。
- 如需修改已有页面，请直接更新对应文件。
- 不建议在仓库中存放敏感信息或动态依赖内容。
- 本仓库主要面向存档与静态展示，不提供后端服务或交互功能。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `docs/` 目录下的 HTML 文件即可查看。
3. 若启用 GitHub Pages，可直接通过仓库 Pages 地址访问对应页面。

## 许可

本仓库内容仅供学习、测试与存档使用。如需引用或分发，请自行确认各页面内容的授权情况。
